Nombre del grupo: FML

Nuestro proyecto consistirá en la integración de las APIs de RocketLeagueStats, Twitter y Telegram Bot. De la API de RocketLeagueStats obtendremos información y estadísticas del videojuego Rocket League tales como rankings o datos individuales del jugador, y estos podrán ser tuiteados por la API de Twitter o enviados via Telegram mediante un bot con su respectiva API.
